# glennhurley.com

Personal website and long-term personal portal.

This repository will start as a simple static site and will (hopefully) evolve over time into a more feature-rich personal platform.

## Current state
- Static HTML/CSS
- Single public landing page
- No build step
- Hosted as a static site

## Planned evolution
- Public blog
- Private journal (authenticated)
- Coding project showcase
- API-backed features

## Tech direction
- Static-first approach
- Backend likely FastAPI when needed
- Authentication for private areas
- Containerized deployment
- CI/CD for deployments

## Structure
- `public/` — Web root (served directory)
- `docs/` — Design notes and roadmap
- `backend/` — Future backend services
- `infra/` — Infrastructure and deployment config

## Status
Early stage. Intentionally simple.
